# Python - Server side rendering

## Description
Rendu de pages côté serveur en Python avec le moteur de templates Jinja2 : introduction, logique de template, lecture de fichiers de données (JSON, CSV) et base de données.

## Fichiers

| Fichier | Description |
|---|---|
| `task_00_intro.py` | Introduction au rendu de templates |
| `task_01_jinja.py` | Utilisation basique de Jinja2 |
| `task_02_logic.py` | Logique conditionnelle dans les templates |
| `task_03_files.py` | Génération de pages à partir de fichiers JSON/CSV |
| `task_04_db.py` | Génération de pages à partir d'une base de données |
| `0-main.py` | Fichier de test |
| `templates/` | Templates Jinja2 utilisés par les scripts |
| `items.json`, `products.csv`, `products.json`, `products.db` | Jeux de données utilisés pour le rendu |
| `template.txt` | Modèle de template texte |
