# Holberton School - Higher Level Programming

Ce repository regroupe des projets de programmation "haut niveau" : bases JavaScript, manipulation du DOM, sérialisation Python, rendu côté serveur, ORM et API REST.

## Sous-dossiers

| Dossier | Description |
|---|---|
| `javascript-warm_up/` | Bases du langage JavaScript |
| `javascript-dom_manipulation/` | Manipulation du DOM en JavaScript |
| `python-serialization/` | Sérialisation de données en Python (JSON, Pickle, CSV, XML) |
| `python-server_side_rendering/` | Rendu de templates côté serveur avec Jinja2 |
| `python-object_relational_mapping/` | ORM Python : MySQLdb puis SQLAlchemy |
| `restful-api/` | Construction d'API REST en Python (Flask) |
