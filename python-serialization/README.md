# Python - Serialization

## Description
Sérialisation et désérialisation de données en Python selon différents formats : JSON, Pickle, CSV et XML.

## Fichiers

| Fichier | Description |
|---|---|
| `task_00_basic_serialization.py` | Sérialisation/désérialisation basique en JSON |
| `task_01_pickle.py` | Sérialisation d'objets Python avec `pickle` |
| `task_02_csv.py` | Lecture et écriture de données au format CSV |
| `task_03_xml.py` | Sérialisation/désérialisation au format XML |
