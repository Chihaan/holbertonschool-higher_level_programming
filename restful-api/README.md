# Python - RESTful API

## Description
Construction et consommation d'API REST en Python : requêtes HTTP, serveur HTTP basique, API Flask et sécurité de base.

## Fichiers

| Fichier | Description |
|---|---|
| `task_02_requests.py` / `main_02_requests.py` | Consommation d'une API avec le module `requests` |
| `task_03_http_server.py` | Serveur HTTP basique en Python |
| `task_04_flask.py` | API REST construite avec Flask |
| `task_05_basic_security.py` | Notions de sécurité de base pour une API (authentification, tokens) |
| `posts.csv` | Jeu de données utilisé par l'API |
